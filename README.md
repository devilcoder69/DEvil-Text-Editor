# DEvil-Text-Editor
This is my first repository.

This is my first python project.

This notepad is basically like a normal text editor in which we can edit or modify our text but it includes some special features like color theme , font color , status bar and you can hide and unhide tool bar and status bar. The program has options such as changing the font, the font size and the font style. The most common use for Notepad is to view or change text (.txt) files and files can be changed in Notepad as well.
